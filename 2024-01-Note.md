## `priority_queue` vs `multiset`

```c++
auto cmp = [](const ListNode* r1, const ListNode* r2){
    return r1->val < r2->val;
};
multiset<ListNode*, decltype(cmp)>pq(cmp);


auto cmp = [](const ListNode* r1, const ListNode* r2){
    return r1->val > r2->val;
};
priority_queue<ListNode*, vector<ListNode*>, decltype(cmp)>pq(cmp);

```
